# NAME

logging::level - display the current logging level

# SYNOPSIS

logging::level

# DESCRIPTION

`logging::level` displays the current script logging level on `stdout`.

It returns success unless a write or assignment error occurs.

# EXAMPLES

    $ logging::level
    6

# SEE ALSO

`logging::setlevel(1)`
