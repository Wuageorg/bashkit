# NAME

readlinkf - display symbolic link target status

# SYNOPSIS

readlinkf *filelink*

# DESCRIPTION

With *filelink* a symbolic link, `readlinkf` resolves paths and displays
the corresponding absolute pathname.

`readlinkf` returns 0 upon success, >0 otherwise.
