# NAME

usleep - micro sleep

# SYNOPSIS

usleep [*n*]

# DESCRIPTION

`usleep` is part of bashkit module `extras`, it waits for a delay
proportional to *n*. When *n* is omitted, it defaults to 1.

# EXAMPLE
    $ usleep 10
