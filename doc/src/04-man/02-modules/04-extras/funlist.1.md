# NAME

funlist - list all declared functions

# SYNOPSIS

funlist

# DESCRIPTION

`funlist` is part of bashkit module extras, it lists all declared functions
by the time of calling.

# EXAMPLE
      $ funlist
      __unhandled
      alert
      bashkit::help
      bashkit::init
      bashkit::load
      bashkit::main
      ...
