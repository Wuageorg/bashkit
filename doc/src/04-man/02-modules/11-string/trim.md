# NAME

string::trim - remove leadind and trailing spaces from string

# SYNOPSIS

string::trim *str*

# DESCRIPTION

`string::trim` removes all leading and trailing spaces from string
*str*.

# EXAMPLE

  $ string::trim '    Hello,  world    '
  Hello,  world
