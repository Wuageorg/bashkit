# NAME

json::from_yaml - convert yaml to json

# SYNOPSIS

json::from_yaml *yaml*

# DESCRIPTION

Given a *yaml* string, `json::from_yaml` converts it to json.

`json::from_yaml` returns 0 upon success, >0 otherwise.

`json::to_yaml(1)`
