# NAME

json::from_toml - convert toml to json

# SYNOPSIS

json::from_toml *toml*

# DESCRIPTION

Given a *toml* string, `json::from_toml` converts it to JSON.

`json::from_toml` returns 0 upon success, >0 otherwise.
